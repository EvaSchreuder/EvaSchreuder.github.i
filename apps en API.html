Apps en API

Een synoniem voor applicaties is apps, dit is de software die zichzelf de toepassingenlaag noemt. Dit zijn dus apps die je vanuit je app store kan downloaden, voorbeelden hiervan zijn whatsapp, instagram of Somtoday.

De API’s zorgen ervoor dat bijna alle apps op alle apparaten kunnen worden geïnstalleerd, dit is zonder dat er voor ieder apparaat een aparte versie nodig is.
